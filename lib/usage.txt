Usage: xa <command>

xa a                set up a temporary vhost
xa set              set xampp modules
xa shortcut         create a shortcut for xampp-control.exe
xa ls [version]     print the information of installed xampp version
xa use <version>    use which version of xampp
xa start            launch xampp-control.exe
xa stop             stop Apache, mySQL, xampp-control.exe

Before run command `xa a` please refer the following link
https://github.com/yanxyz/xampp-windows-cli/blob/master/config-sample/httpd-temp.conf

Example:
  xa a
  shorthand for `xa a --vhost="a.com www.a.com;./"`
  set up a temporary vhost: a.com(./)

  xa ab
  shorthand for `xa a --vhost="a.com www.a.com;./a" --vhost="b.com www.b.com;./b"`
  set up a.com(./a) and b.com(./b)

  xa set [vhosts]
  replace apache/conf/extra/httpd-vhosts.conf with custom httpd-vhosts.conf
  and restart xampp
